### Neurons

A Neuron is the fundamental unit of a Neural Network. Think of it as a tiny decision maker.

It works just like a biological neuron. In an artificial neuron: first it takes inputs, then a weighted sum is applied, then an activation function is applied, and finally we get the output.

### Weights

A weight is applied on each input to define how important that input is.

Let's say we have 2 inputs — exam score and interview score. Maybe exam score matters more, so we assign w1 = 0.8 and w2 = 0.2. That means 80% weightage goes to exam score and 20% to interview score.

### Activation Function

Without an activation function, a Neural Network is just a fancy linear equation.

Activation functions introduce **non-linearity**, which allows deep learning models to learn faces, speech, language, objects, and a lot more.

Some famous activation functions:
- Sigmoid
- ReLU
- Softmax

### Layers

One neuron alone isn't powerful. We need to combine many neurons together.

A collection of neurons is called a **Layer**.

There are mainly 3 types of layers:
- Input Layer
- Hidden Layer
- Output Layer
